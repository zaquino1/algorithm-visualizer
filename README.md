# Algorithm Visualizer

An interactive tool for learning and understanding algorithms through visualization.

## Features

- **Sorting Algorithms**: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, Heap Sort
- **Searching Algorithms**: Linear Search, Binary Search
- **Interactive Controls**:
  - Step-by-step execution
  - Adjustable animation speed
  - Custom array input
  - Random array generation
- **Performance Metrics**:
  - Real-time comparison and swap counters
  - Time complexity display
  - Execution time tracking

## Getting Started

1. Clone the repository
2. Open in IntelliJ IDEA or your preferred IDE
3. Run `org.algovisualizer.Main`

## Usage

1. Select an algorithm from the menu or dropdown
2. Generate a random array or input custom values
3. Click 'Run' to start visualization or 'Step' for manual control
4. Adjust speed using the slider
5. View statistics in the bottom panel
